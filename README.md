Git2JSS
===============================

version number: 0.0.1
author: Geoff Lee

Overview
--------
*********************** Unstable! Do not use! ************************
Push scripts from a Git repo to a JSS. Includes templating and tagging.

Installation / Usage
--------------------

To install use pip:

    $ pip install git2jss


Or clone the repo:

    $ git clone https://github.com/gkluoe/git2jss.git
    $ python setup.py install
