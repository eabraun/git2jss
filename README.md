Git2JSS
===============================

version number: 0.0.3
author: Geoff Lee

Overview
--------
Push scripts from a Git repo to a JSS. Includes templating and tagging.

Installation / Usage
--------------------

Clone the repo:

    $ git clone https://github.com/gkluoe/git2jss.git
    $ python setup.py install
